# Modulo
Modulo - An esoteric language I made based on maths.
# How it works
You can add 3 + 4 with this line of code :
```
3 4 + .
```
You can then multiply 7 by 12 using this line :
```
7 12 * .
```
It also has the functions of subtracting, dividing and making something 1% of itself!
# The Unicode function
Everything I just mentioned is all fine and dandy, BUT this is the coolest function in Modulo!
```
85 ` 110 ` 105 ` 99 ` 111 ` 100 ` 101 ` 33 ` 10 `
```
The ` function has the capability of turning any number into it's Unicode equivalent!
For example :
- A = 65,

- B = 66,
  
And so on...
# Binary & Hexadecimal Support
This code works in Modulo :
```
0b11 0b100 + .
```
Yes, it's Binary in Modulo!
This also works :
```
0x3 0x4 * .
```
And yes, that is Hexadecimal, again, in Modulo!
Modulo supports both Base-2 and Base-16!
# Comments
The comments are basically just adrenaline to feel like a coder back in 1946.
```
85 ` 110 ` 105 ` 99 ` 111 ` 100 ` 101 ` 33 ` 10 ` 0xDEADBEEF It prints 'Unicode!'.
```
Noticed anything?

The function 0xDEADBEEF is the comment function!

Just to compare it to other languages :

Python : #

C : //

Hexadecimal : 0x90
